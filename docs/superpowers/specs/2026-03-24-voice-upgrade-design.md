# 朗读器 v2.0 语音升级设计文档

**日期：** 2026-03-24
**项目：** tts-reader
**版本：** 1.0 → 2.0

---

## 一、需求概述

在现有朗读器基础上升级语音功能：

1. **接入讯飞 TTS**：用高质量神经语音替代系统 TTS，朗读更自然
2. **保留系统语音**：作为兜底，配额耗尽或无网络时自动切换
3. **两级音色选择**：先选风格分类，再选具体音色
4. **讯飞账号自助管理**：用户自行在讯飞开放平台注册、充值，在 app 内填入凭证
5. **界面全中文化**：所有 UI 文案换成中文

---

## 二、整体架构

```
Electron 主进程
  - 文件解析（Word / PDF / TXT + OCR）
  - 讯飞凭证读写（config.json，存于 app userData）
  - IPC 通道

渲染进程
  ├── TTS 引擎管理器
  │     ├── 讯飞引擎（WebSocket + Web Audio API）
  │     └── 系统引擎（Web Speech API，原有逻辑）
  ├── 播放器 UI（底部控制栏扩展版）
  └── 设置面板
```

TTS 引擎管理器对外暴露统一接口：`speak(text, voice)` / `pause()` / `resume()` / `cancel()`，切换引擎不影响播放器逻辑。

## 三、讯飞 WebSocket 引擎

### 3.1 鉴权

讯飞 WebSocket TTS 使用 HMAC-SHA256 签名鉴权，每次连接动态生成带签名的 URL。所需凭证：AppID、APIKey、APISecret，由用户在设置面板填入，保存到本地 `userData/config.json`。

### 3.2 逐句朗读流程

```
取当前句文本
  → 生成签名 WebSocket URL
  → 建立连接，发送合成参数（音色代码、语速、音量）
  → 分块接收 Base64 编码 PCM 音频
  → Web Audio API 实时解码播放（AudioContext + BufferSourceNode）
  → 收到 status=2（结束帧）→ 关闭连接 → 触发下一句
```

### 3.3 合成参数

```json
{
  "vcn": "xiaoyan",
  "speed": 50,
  "volume": 50,
  "pitch": 50
}
```

语速由 UI 滑块（0.5x–2.0x）线性映射到讯飞 speed（0–100）。

### 3.4 回退机制

讯飞连接失败时（网络异常、凭证无效、配额耗尽）：
- 自动切换至系统语音继续当前句朗读
- 顶部显示提示条：「讯飞连接失败，已切换系统语音」

## 四、音色体系

### 4.1 预设分类（讯飞标准音色）

| 分类 | 音色代码 | 显示名称 |
|------|---------|---------|
| 温柔女声 | xiaoyan / aisjiuxu / x4_lingfeijing / x4_yezi | 晓燕 / 晓萌 / 灵飞晶 / 叶子 |
| 磁性男声 | aisbabyxu / x4_lingfeijun / x4_kele | 小炫 / 灵飞峻 / 可乐 |
| 新闻播报 | x4_lingfeixuan / aisjingjing | 灵飞璇 / 晶晶 |
| 儿童声 | x4_xiaomei / x4_xiaobei | 小美 / 小贝 |
| 系统语音 | — | 调用 Web Speech API，列出系统已安装的中文语音 |

### 4.2 两级选择交互

点击底部控制栏「音色」按钮 → 面板向上弹出：
- **第一级**：分类标签横排（温柔女声 / 磁性男声 / 新闻播报 / 儿童声 / 系统语音）
- **第二级**：当前分类下的音色按钮，点击立即生效并关闭面板

切换引擎（讯飞 ↔ 系统）时，第二级音色列表自动切换对应内容。


### 5.2 设置面板

右上角齿轮图标打开，覆盖层展示：

```
┌──────────────────────────────────────┐
│  讯飞语音设置                         │
│  AppID:     [__________________]     │
│  APIKey:    [__________________]     │
│  APISecret: [__________________]     │
│  [去讯飞开放平台注册 ↗]   [保存]      │
│  状态：● 已连接 / ○ 未配置            │
└──────────────────────────────────────┘
```

保存后立即用当前凭证发起测试连接，更新状态指示。

### 5.3 中文化清单

全部 UI 文案改为中文，包括但不限于：

- 打开文件、未打开文件
- 回到开头、上一句、播放、暂停、下一句、跳到结尾
- 语速、设置、保存、取消
- 正在识别扫描版 PDF，请稍候…
- OCR 识别中：第 X / Y 页…
- 文件解析失败、OCR 识别失败
- 讯飞连接失败，已切换系统语音

## 六、数据存储

凭证保存至 Electron `app.getPath('userData')/config.json`：

```json
{
  "xunfei": {
    "appId": "",
    "apiKey": "",
    "apiSecret": ""
  },
  "lastEngine": "xunfei",
  "lastVoice": "xiaoyan"
}
```
